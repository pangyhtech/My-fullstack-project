## Features
- User authentication & profiles
- Shopping cart & checkout
- Membership tiers & points
- Order history & favorites
- Product reviews
- 30+ screens

## Tech Stack
- SwiftUI
- Python (Backend)
- MVVM Architecture

## Quick Start
1. Start backend server:
```bash
cd server
python3 server.py
```

2. Open Xcode project:
```bash
open sweetspro.xcodeproj
```
