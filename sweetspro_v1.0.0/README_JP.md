## 概要
AI支援により6時間で開発された、洋菓子・和菓子専門の本格的なiOSショッピングアプリ。

## 機能
- ユーザー認証とプロフィール
- ショッピングカートと決済
- 会員ランクとポイント
- 注文履歴とお気に入り
- 商品レビュー
- 30以上の画面

## 技術スタック
- SwiftUI
- Python（バックエンド）
- MVVMアーキテクチャ

## クイックスタート
1. バックエンドサーバーを起動:
```bash
cd server
python3 server.py
```

2. Xcodeプロジェクトを開く:
```bash
open sweetspro.xcodeproj
```

## 統計
- ⏱️ 開発時間: 6時間
- 📱 画面数: 30以上
- 💻 コード行数: 約5,000行
